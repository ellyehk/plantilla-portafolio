Página promocional para un evento académico, con funcionalidades como cuenta regresiva, detalles de ponentes, cronograma de actividades y formulario de registro en línea.

Landing page para promocionar un evento especial (2024)

Se diseñó un sitio limpio y funcional en Figma, priorizando una estructura que facilitara la navegación por secciones: objetivos, ponentes, programa, ubicación y formulario de inscripción. Se utilizó Bootstrap y JavaScript para agregar interactividad como el contador y desplazamiento suave. El formulario se conectó a Google Forms para capturar inscripciones fácilmente.

Detalles clave:
  • Sección de cronograma con diseño vertical u horizontal
  • Contador regresivo con animación
  • Bloques informativos por ponente (nombre, tema, horario)
  • Inscripción directa mediante Google Forms
  • Responsive Design para móvil y escritorio
  • Integración con calendarios
