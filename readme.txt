# 🌐 PRUEBAWEB

Este es un proyecto web que muestra un **portafolio de proyectos** con tarjetas dinámicas y modales. Cada tarjeta carga información adicional desde archivos HTML individuales, permitiendo una navegación fluida y modular.

## 🚀 Características

- **Diseño responsivo** con Bootstrap 5.  
- **Carga dinámica de contenido** en modales.  
- **Efectos interactivos** con Swiper.js.  
- **Organización modular** de imágenes y archivos HTML.  
- **Compatible con GitHub Pages o cualquier hosting**.  

---

## 🛠️ Tecnologías utilizadas

- **HTML5**  
- **CSS3 (Bootstrap 5)**  
- **JavaScript (Vanilla)**  
- **Swiper.js** para efectos interactivos  
- **Figma** (para prototipos de diseño)  

---

## 📂 Estructura de carpetas

```plaintext
